/ontario-cpr-society

Led the frontend development for Ontario CPR Society’s website launch, a platform offering CPR and first aid training courses. The client needed a modern and high-converting website to showcase their various packages effectively and improve user engagement. My development work involved translating Figma design prototypes into high-quality code ensuring the webpages were interactive, responsive and accessible. I built 10+ modular and reusable components using React.js, Gatsby and Bootstrap, reducing redundancy and improving maintainability. I structured the website by storing section-specific data in JSON-based data structure, enabling dynamic content rendering and reducing content update time by 50%. This approach accelerated development time, enhanced performance and simplified future content updates, allowing for rapid scalability as new packages or services were added.
